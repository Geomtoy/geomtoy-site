Geomtoy

A 2D geometry responsive computing, visualizing and interacting library.

Features

Core

Basic geometries and general geometries with usual functionality out of the box.

Microtask based event system handling geometric associations.

Browser independent and can run in all javascript environments.

Multiple predicates for intersections between basic geometries.

Application of affine transformations returning the transformed geometry.

Inversion of points, lines, and circles.

Full boolean operations respecting fill rules and area computation of complex general geometries.

View

Interaction framework including most of the functionality.

Microtask based rendering without user attention.

Support both SVG and Canvas as renderers.

Customizable boundless interface of the coordinate system.

Support specifying the coordinate system and display status

View scope and view element scope interaction events.

Support touch devices.

Installation

Via CDN


<!-- All-in-one bundle package -->       
<script src="https://cdn.jsdelivr.net/npm/geomtoy@{{version}}/dist/index.min.js"></script>
<script src="https://unpkg.com/browse/geomtoy@{{version}}/dist/index.min.js"></script>

<!-- Scoped packages -->
<!-- @geomtoy/util -->
<script src="https://cdn.jsdelivr.net/npm/@geomtoy/util@{{version}}/dist/index.min.js"></script>
<script src="https://unpkg.com/browse/@geomtoy/util@{{version}}/dist/index.min.js"></script>

<!-- @geomtoy/core -->
<script src="https://cdn.jsdelivr.net/npm/@geomtoy/core@{{version}}/dist/index.min.js"></script>
<script src="https://unpkg.com/browse/@geomtoy/core@{{version}}/dist/index.min.js"></script>

<!-- @geomtoy/view -->
<script src="https://cdn.jsdelivr.net/npm/@geomtoy/view@{{version}}/dist/index.min.js"></script>
<script src="https://unpkg.com/browse/@geomtoy/view@{{version}}/dist/index.min.js"></script>

Via npm


# Install all-in-one bundle package
npm i geomtoy

# Install scoped packages
npm i @geomtoy/util
npm i @geomtoy/core
npm i @geomtoy/view
